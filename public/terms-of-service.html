1. Service Description

The Datadog APM Azure Vending Machine ("the Service") is a self-service web application that enables users to automatically deploy Datadog Application Performance Monitoring (APM) configurations to their Azure App Services using Azure Resource Manager (ARM) templates.

The Service provides:

  • Azure Active Directory authentication integration
  • ARM template deployment for Datadog APM configuration
  • Self-service resource provisioning in your Azure subscription
  • Automated configuration of Datadog monitoring extensions
